# Plataforma Web ONG — Entrega (Tema Verde)

Estrutura e arquivos incluídos:
- `index.html`, `projetos.html`, `cadastro.html`
- `css/styles.css` (tema verde)
- `js/main.js` (máscaras e validação)
- `images/` (imagens otimizadas em JPG e WebP)
- `docs/README.md` (este arquivo)
- `validator/relatorio-validacao.html` (modelo de relatório)

Instruções:
1. Extraia o ZIP.
2. Abra `index.html` localmente para testar.
3. Para validação W3C: suba cada HTML em https://validator.w3.org/ e corrija conforme necessário.

Observações:
- Formulário realiza validação nativa HTML5 e validações JS complementares.
- Em produção, valide e sanitize todos os dados no servidor.
